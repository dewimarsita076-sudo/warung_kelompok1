# warung_kelompok1
Aplikasi ini dirancang untuk membantu pemilik warung dalam mencatat pesanan pelanggan, mengelola stok menu, menghitung tagihan secara otomatis, serta menyimpan riwayat transaksi tanpa menggunakan database ataupun GUI. Sistem dibuat sepenuhnya berbasis class untuk memenuhi implementasi konsep OOP secara terstruktur dan modular.
